# personal-site

Mi portafolio personal. Hecho con Angular 18, desplegado en GitHub Pages.

**Stack:** Angular 18 · TypeScript · Angular Router · CSS puro · angular-cli-ghpages

---

## Correr en local

```bash
npm install
npm start
```

Abre `http://localhost:4200`.

## Desplegar en GitHub Pages

```bash
npm run build:ghpages
npx angular-cli-ghpages --dir=dist/personal-site/browser
```

En GitHub: Settings → Pages → Branch `gh-pages` → `/ (root)`.
